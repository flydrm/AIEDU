## 项目特点
- **目标用户**：自家3岁男孩
- **开发周期**：2周
- **技术栈**：Android原生 + AI服务
- **内容规模**：20-30张教育卡片
- **核心功能**：AI内容生成、语音合成、播放互动、进度保存
- **设备支持**：手机和平板响应式布局
- **AI增强**：智能内容生成、童声TTS、学习建议
