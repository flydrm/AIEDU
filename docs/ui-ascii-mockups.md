```
╭─────────────────╮
│                 │ ← 阴影效果
│    下 一 张      │

```

```
┌─────────────────┐
│    下 一 张      │
```

## ASCII 色彩标注

方案一配色：
```
█ = #FF4444 (主红色)
▓ = #FFB5B5 (浅红色)
░ = #FFF5F5 (背景粉)
□ = #FFFFFF (纯白)
■ = #333333 (文字黑)
```

方案二配色：
```
█ = #FF0000 (纯红)
■ = #000000 (纯黑)
□ = #FFFFFF (纯白)
░ = #F5F5F5 (浅灰)
```

这两套ASCII设计展示了两种截然不同的设计风格，您可以直观地看到界面布局和交互元素的差异。
